# Loan Application Prediction
*Machine learning model to predict loan defaults and automate eligibility with 75% accuracy.*
* Analyzed **10,000+ loan records** using **EDA**, identifying key factors like **Credit History** and **Balance Income.**
* Trained **Logistic Regression, Decision Tree, and Random Forest**, achieving an **F1-score of 0.72.**
* Improved risk assessment by detecting **loan defaulters with 75% accuracy**, reducing default risks.
